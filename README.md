### vue+vuex+elementUI+axios
* assets 静态文件
* compontes 组件包括 登录/注册 文件上传 图片预览 柱/饼/线图表
* core 拦截器 路由守卫 i18n
* layout 导航栏
* plugins 插件 通用方法
* router 路由
* store vuex管理
* style 通用样式
* view 页面
* APP.vue 主入口
* main.js
* vue.config.js 配置文件
